# MeshBoolean
Make Boolean Operator on Mesh. In Unity.

对网格做布尔操作（合集、交集、差集）。有两种途径，一是操作网格，二是利用Shader达到看起来像那么回事的效果。

# Versions
Unity 5.4.0f3
